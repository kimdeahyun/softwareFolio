```

아르바이트 형식으로 다니게 되었습니다. 거기서의 역할은 햄 배합 하는 곳에서
제가 일을 하게 되었습니다. 그 곳에서 얻을수 있었던건 끈기였습니다. 
정말 하기 싫고 힘들고 야간도 했었지만 단지 하나의 목표 때문에 돈을 벌어서
```
